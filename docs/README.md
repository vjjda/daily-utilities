# Tài liệu Công cụ Dòng lệnh

Dưới đây là danh sách các công cụ dòng lệnh (command-line tools) của dự án `daily-utilities`.

---

### [`btool`](./tools/btool.md)

- **B**ootstrap **Tool**: Tự động tạo cấu trúc file cho một công cụ mới từ file đặc tả.

### [`cdiag`](./tools/cdiag.md)

- **C**lipboard **Diag**ram: Xử lý code đồ thị (Graphviz/Mermaid) từ clipboard và render ra ảnh.

### [`cpath`](./tools/cpath.md)

- **C**heck **Path**: Kiểm tra và sửa các comment đường dẫn (`# Path: ...`) ở đầu file.

### [`ctree`](./tools/tree.md)

- **C**ustom **Tree**: Hiển thị cây thư mục với các tùy chọn lọc nâng cao.

### [`forc`](./tools/forc.md)

- **For**mat **C**ode: Định dạng (format) các file mã nguồn, hỗ trợ chế độ gia tăng.

### [`ndoc`](./tools/ndoc.md)

- **N**o **Doc**string: Loại bỏ docstring và comment khỏi các file mã nguồn.

### [`pcode`](./tools/pcode.md)

- **P**ack **Code**: Đóng gói nhiều file mã nguồn thành một file context duy nhất cho LLM.

### [`sgen`](./tools/sgen.md)

- **S**tub **Gen**erator: Tự động tạo file stub `.pyi` từ các file `__init__.py`.

### [`zrap`](./tools/zrap.md)

- **Z**sh W**rap**per: Gói một script Python thành một file thực thi Zsh tự quản lý venv.
